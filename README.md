\## âœ¨ Project Overview



The Employee Management System (EMS) helps organizations efficiently manage employees and tasks through a role-based system with separate dashboards for admins and employees.



---



\## â­ Key Features



\- Full-stack Employee Management System developed using MERN stack  



\- Role-based authentication using JWT and bcrypt for secure admin and employee login  



\- Admin Dashboard to:

&nbsp; - Add and delete employees  

&nbsp; - Assign tasks with deadlines and priority levels (High, Medium, Low)  



\- Employee Dashboard to:

&nbsp; - View assigned tasks  

&nbsp; - Update task status as Completed, Pending, or Failed in real time  



\- Secure backend APIs with protected routes
